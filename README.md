## zsh my custom using oh-my-zsh

Base Themes is wedisagree.

### Setup
----------------

1. install [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)

2. cd your-dir

3. `git clone git://github.com/SenZhangAI/oh-my-zsh-sen.git`

4. `cd oh-my-zsh-sen`

5. `./install.sh`

6. `source ~/.zshrc`
